峰会简介

7月30日，由澳财ALLFIN主办、博满金资集团协办的 “新格局，新挑战，新机遇”2019澳财投资高峰论坛，在悉尼凯悦酒店隆重举行。论坛邀请到了澳大利亚顶级金融机构和行业领袖，帮助与会的350位华人高净值投资者在机遇与挑战并存的当下，为未来投资之路找到最有价值的助力。

第三部分 : [ ALLFIN ROADSHOW ] 对接澳洲领先项目资源

SalesPreso

SalesPreso是一家基于软件即服务（SaaS）的营销展示平台，提供企业销售团队个性化营销展示功能。通过整合大量客户相关数据，SalesPreso可自动化生成由数据驱动的多维业务展示界面，节约制作时间，使企业获得更多扩大业务机会，增加收入。近年来，公司客户数猛增，当前客户包括Telstra、可口可乐、News Corp和FlightCentre（美国，欧洲，印度）等澳大利亚本土及全球企业。

Crimson Education
红序教育集团

Crimson Education是世界名校申请专家。区别于传统面对面咨询，集团利用线上平台参与学生学校申请全过程，打破地域壁垒，擅长管理和激励学生，并为他们个性化定制最合适的计划，最大可能地提高他们进入顶级名校的机率。CrimsonEducation线上拥有2000多名指导教师及顾问组成的专业团队，成员均为英美最顶尖大学的学生或校友，拥有学生渴望了解的关于学院的学习和申请的知识和经验。

Identitii

Identitii（ASX：ID8）察觉到金融市场信息不通畅、信息遗失量大的问题，创建了基于区块链技术的交互平台，其网络覆盖所有支付系统或服务，使供应商、买家、银行和监管机构得以实现信息代币化，并能及时共享详尽的支付信息。目前全球十大银行之一的汇丰银行，与identitii签署了关于软件平台Overlay+的使用协议。

FIIG Securities

成立于1998年，FIIG Securities是澳大利亚规模最大的固定收益专家，帮助个人投资者、投资顾问、非盈利组织和机构管理其固定收益类投资。FIIG资管量超过80亿澳元，拥有7000余名投资客户，提供400多种国内外企业债券投资机会，为50多家企业发行了企业债券，共募资超过20亿澳元。FIIG在布里斯班、悉尼、墨尔本和珀斯均设有办公室，员工数超过130人，旨在协助投资人获取固定收益投资机会和专业知识。

Homart Pharmaceuticals
活曼特药业

成立于1992年，活曼特药业是澳洲高端营养保健品、美容护肤品、母婴乳制品的研发、生产及销售商。集团拥有顶尖研发团队和生产设备，并获得TGA（澳大利亚药品管理局认证）、GMP（生产质量管理规范）和FDA（食品药品监督管理局认证）的认证。迄今，集团销售足迹遍布全球50多个国家和地区，是2008年奥运会、2010年亚太经合组织、2016年G20峰会的独家指定澳洲健康产品合作伙伴。

Shareable Apps

Shareable Apps是由Amplifier Corporation Pty Limited创建的全球首个非传统应用商店模式的应用开发和分发平台——无需成为专业软件开发人员，即可通过平台创建适用于各种设备的应用软件。平台拥有超过20,000名应用程序创建者，使用者中包括澳大利亚联邦政府、7-Eleven、SigmaPharmaceuticals、红十字会、CSR、悉尼艺术节和Blackmores等企业客户。

Australia Bee Blockchain Foundation
澳大利亚小蜜蜂集团

澳大利亚小蜜蜂集团成立于2017年，总部设于悉尼，专注于金融、科技研发、新零售三大板块，是一家拥有澳洲金融全牌照的投资集团。集团拥有广泛的区块链行业资源，丰富的数字资产管理经验，高效的投资交易团队和资深的顾问团队，是澳大利亚区块链产业园的创始成员，在全球拥有多家成熟的数字资产交易所，及数字资产管理公司。

Ezifin

Ezifin是LakebaGroup金融服务体系五大服务之一。公司通过提供稳定现金流帮助客户巩固现有业务，并提供技术支持，使客户得以体验“智能银行”的便捷；技术平台EziTech致力于为客户提供全方位服务，以提高客户价值、减少客户流失。Ezifin目前筹备建立EziBank，将通过降低资金成本来使客户利益最大化。
